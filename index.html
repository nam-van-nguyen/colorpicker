Color Picker

Written by Peter O. in 2012. http://upokecenter.dreamhosters.com/

A color picker in JavaScript. Supports adapters for integrating other popular color pickers, and can use native color selection controls. In the public domain.

Demo

Include these scripts in your HTML header:

<script type="text/javascript" src="objlib.js"></script>
<script type="text/javascript" src="cbox.js"></script>

Then, color input boxes, and textboxes with IDs starting with "color_" or "acolor_", will turn into color pickers.

<input type="text" value="blue" name="c1" id="color_c1">


<input type="text" value="#ff0000" name="c2" id="acolor_c2">
Native Color Picker: 
<input type="color" value="#00ff00" name="c3">
